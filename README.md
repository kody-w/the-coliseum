# The Coliseum 🏛️

An evolutionary arena where AI **creatures** — defined by trait-vector genomes — compete at
real challenges, **breed** when they win, and **die** when they lose. Built in parallel by a
brainstem flock: one machine forges the bloodlines, one runs the arena, one builds the floor
you watch it on.

## The loop
1. Seed a population of creatures (random genomes).
2. The arena pits them at challenges; a creature's genome shapes how it fights.
3. Winners breed (crossover + mutation); the bottom is culled.
4. Repeat for N generations. Watch lineages rise and fall on the Coliseum Floor.

See **SPEC.md** for the data contract every part builds against.
